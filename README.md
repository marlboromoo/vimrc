# Marlboromoo's vimrc
Personal settings/plugins for vim/gvim.

## Installation
### With Vundle
```
cd ~/
git clone http://marlboromoo@github.com/marlboromoo/vimrc.git
bash ./vimrc/init.sh
vim +BundleInstall +qall
cd ~/.vim/bundle/YouCompleteMe
./install.sh --clang-completer
```
